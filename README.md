# 🐱 CatStore

CatStore is a frontend web application built with HTML, CSS, and vanilla JavaScript.  
It simulates a simple cat-themed online store where users can browse cats, view details, and manage a shopping cart.

---

## Features

- Browse a collection of cats
- View detailed information about each cat
- Add and remove items from a shopping cart
- Multi-page structure (Home, Cats, Cart, Owner)
- Reusable navigation bar component
- Clean and modular file structure

---

## Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
