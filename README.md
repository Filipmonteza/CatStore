# 🐱 CatStore

CatStore is a frontend web application built with HTML, CSS, and vanilla JavaScript.  
The application simulates a simple cat-themed online store where users can browse different cats, view details, and manage a shopping cart.

---

## 🚀 Features

- Browse a collection of cats
- View detailed information about each cat
- Add and remove items from a shopping cart
- Multi-page application (Home, Cats, Cart, Owner)
- Reusable navigation bar component
- Modular JavaScript structure
- Clean and organized file architecture

---

## 🛠️ Tech Stack

- **HTML5** – Structure and content
- **CSS3** – Styling and layout
- **JavaScript (Vanilla JS)** – Logic and interactivity
